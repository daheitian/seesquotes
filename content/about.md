# 💡 这是什么？

**小信号**是一个个人数字花园，用于收集、整理和分享那些容易被忽视但价值巨大的"小信号"。

> "在噪声中寻找信号，在信号中发现意义。"
> — 小信号的设计理念

## 🗂️ 三个板块

### 💭 见室引号
收集金句、想法和优质信息源。来自 Notion 数据库的自动同步，每3小时更新一次。

### ✍️ 写作
更长形式的思考与文章。基于 GitHub Issues 管理，自动同步到站点。

### 📚 阅览
正在读、想读和已读的书单。追踪阅读进度，记录读书笔记。

## ⚙️ 技术架构

- 静态 HTML + Vanilla JS
- GitHub Pages 托管
- Notion API 数据同步
- GitHub Issues 写作管理
